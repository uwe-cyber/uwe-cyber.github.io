IoT HomeLab

IoT HomeLab is a project that was developed in 2020, to continue our outreach activity when the Covid-19 pandemic hit. This activity allows students to deploy a hosted CTF web server that can be linked up to control IoT devices in the home. The system is designed to work specifically with devices that utilise the Tuya IoT network (commonly via port 6868). This includes many IoT light bulbs, power switches, as well as more elaborate devicecs such as IoT robot vacuum devices, and humidifier devices.
