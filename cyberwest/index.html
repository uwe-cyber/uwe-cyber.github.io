CyberWEST - West of England Schools and Teachers

UWE Bristol lead the CyberWEST project, supported by the National Cyber Security Centre in collaboration with the University of Bristol, University of Exeter and the University of Plymouth.

The project builds on previous outreach to develop schools workshops across the South West region, accompanied by an activity repository and an equipment loan library to support teaching and learning in Cyber Security Education.

The main repository for workshop activities is available here: https://github.com/uwe-cyber/teachersworkshop2024

Below is a searchable set of activities, covering both the recent workshops, previous workshop activities, and other useful external resources, which we hope is useful for signposting the wealth of activities that are available online.

To learn more about the project, please email phil.legg@uwe.ac.uk.
