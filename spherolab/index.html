SpheroLab

SpheroLab is an activity that was firsted delivered as part of the NCSC EmPowerCyber event held at University of Gloucestershire in 2019. The activity allows students to control a Sphero robot through Python scripting language. The communications with the device are managed via Bluetooth. Code can be executed to move the robot, as well as control lights and sounds (subject to Sphero model functionality).
